SES EFEKTLERİ
=============

Procedural sesler — gerçek kayıt değil, ama kulak yormayacak şekilde ayarlandı:

  Gökyüzü    → hafif rüzgar + seyrek yumuşak pırıltı (4–9 sn arası)
  Konser     → sürekli kalabalık uğultusu + ara sıra uzun tezahürat dalgası
  Telefon    → mesaj bildirimi + yazıyor tık
  Meyhane    → yumuşak def ritmi
  Açık hava  → konser gibi kalabalık

GERÇEK SES İSTERSEN
-------------------
assets/audio/ klasörüne MP3 koy:

  crowd.mp3   → konser / açık hava kalabalığı (loop)
  wind.mp3    → gökyüzü rüzgarı (loop)
  mey.mp3     → açık hava şarkısı

Dosya varsa procedural yerine o çalar (ileride eklenebilir).
Şimdilik procedural kullanılıyor.
